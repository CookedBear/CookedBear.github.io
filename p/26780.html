Python 爬虫学习


Anaconda 与 Jupyter 配置

Anaconda 为不同的 Python 项目运行创建了相互独立的虚拟环境，他们分别对自身环境进行包管理。

创建环境并激活后使用 pip install jupyter 在对应环境中安装 jupyter。没有自动弹出窗口、设置工作路径、显示中文菜单等可参考这篇文章。
